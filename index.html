Perguntas Frequentes

O desenvolvedor responsável pelo Kwanza Game Studio é angolano, nascido e criado em Angola. Toda a inspiração para os projetos vem da rica cultura, história e diversidade do país, buscando sempre valorizar e representar elementos autênticos da identidade angolana nos jogos desenvolvidos.

O estúdio é liderado por TonilsonJM, nome artístico de António Monteiro. Com paixão por tecnologia, arte e cultura, TonilsonJM atua como desenvolvedor solo, responsável por todas as etapas do processo criativo, desde a concepção das ideias até a programação, arte e publicação dos jogos. Sua missão é criar experiências inovadoras que dialoguem com o público angolano e internacional.

Até o momento, nenhum dos projetos desenvolvidos pelo Kwanza Game Studio recebeu financiamento externo, patrocínio ou investimento. Todo o trabalho é realizado de forma independente, com recursos próprios, o que reforça o compromisso e a dedicação do desenvolvedor em entregar produtos de qualidade mesmo diante de desafios financeiros.

Não. Todo o desenvolvimento dos jogos é realizado de forma individual por TonilsonJM. Desde a programação, design, modelagem 3D, animação, até a trilha sonora e testes, cada etapa é conduzida pessoalmente, garantindo uma visão única e autoral em cada projeto lançado pelo estúdio.

O processo de desenvolvimento envolve o uso de diversas ferramentas profissionais, como Unity para programação e engine de jogos, Visual Studio Code para codificação, Blender para modelagem e animação 3D, Photoshop para edição de imagens, DragonBones para animações 2D, entre outras soluções que auxiliam na criação de experiências ricas e inovadoras.
